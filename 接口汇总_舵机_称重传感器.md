# 舵机 & 称重传感器 — 接口汇总

> STM32H7R3 开发板 | 裸机 | SRAM运行 | DAPLink调试

## 一、SG90 舵机

### 接线

| 舵机线色 | 连接 | 说明 |
|---------|------|------|
| 🟠 橙/黄 (信号) | **PA6** (P2排针) | PWM信号 |
| 🔴 红 (电源) | **5V 独立供电** | 不可从3.3V取电 |
| 🟤 棕 (地) | **5V GND + 开发板 GND** | 必须共地 |

### PWM参数

| 参数 | 值 |
|-----|-----|
| 频率 | 50Hz (20ms周期) |
| 0° 脉宽 | 700us |
| 120° 脉宽 | ~1633us |
| 180° 脉宽 | 2100us |

### 行为

上电从 0° 缓摆到 120°，再摆回，循环往复。

### 代码

- 主程序: `firmware/module_tests/servo_test/sg90_servo_pwm.c`
- 说明: `firmware/module_tests/servo_test/舵机测试说明.md`

## 二、HX711 称重传感器 (三路)

### 接线

| | 传感器1 | 传感器2 | 传感器3 |
|------|---------|---------|---------|
| **DOUT** | **PB6** | **PB8** | **PA4** |
| **SCK** | **PB7** | **PB9** | **PA7** |
| **VCC** | DAPLink 3.3V | DAPLink 3.3V | DAPLink 3.3V |
| **GND** | DAPLink GND | DAPLink GND | DAPLink GND |

### 称重传感器 → HX711绿端子

| 传感器线 | HX711端子 |
|---------|-----------|
| 红线 (激励+) | **E+** |
| 黑线 (激励-) | **E-** |
| 白线 (信号-) | **A-** |
| 绿线 (信号+) | **A+** |

> 颜色以实物标注为准

### 协议

- HX711: 24bit ADC，通道A增益128
- DOUT拉低 = 转换完成
- 主机输出25个SCK脉冲读取数据
- 第25脉冲设置增益

### 零点标定

上电空载自动标定：丢弃前6个不稳定读数，取3次平均作为零点。

### 屏幕显示 (药品页面)

| 字段 | 传感器 |
|------|--------|
| **W1** | 传感器1 (PB6/PB7) |
| **W2** | 传感器2 (PB8/PB9) |
| **W3** | 传感器3 (PA4/PA7) |

### 代码

- 主程序: `firmware/module_tests/hx711_test/main_hx711.c`
- 说明: `firmware/module_tests/hx711_test/README.md`
- 编译: `make -f Makefile_hx711`
- 加载: `python load_to_sram.py`

## 三、H7RS 芯片关键注意事项

| 事项 | 说明 |
|------|------|
| RCC_AHB4ENR地址 | **+0x140** (非标准H7的+0x50) |
| GPIO基地址 | GPIOA=0x58020000, GPIOB=0x58020400, GPIOD=0x58020C00 |
| Flash烧录 | ❌ 暂不支持 (芯片安全限制) |
| SRAM运行 | ✅ 需手动清零BSS段 |
| 调试器GPIO写入 | ❌ AHB-AP无法写GPIO，须CPU代码操作 |
