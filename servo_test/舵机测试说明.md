# 舵机测试 (SG90/MG90S)

基于 STM32H7R3 裸机，软件模拟PWM驱动舵机。

## 接口定义

| 舵机线色 | 连接目标 | 说明 |
|---------|---------|------|
| 🟠 橙/黄 (信号) | **PA6** | GPIOA Pin 6, 开发板排针引出 |
| 🔴 红 (电源) | **5V 独立供电正极** | ⚠️ 不能从开发板3.3V取电 |
| 🟤 棕 (地) | **GND (共地)** | 5V电源负极 + 开发板GND相连 |

### 开发板引脚位置

- PA6: P2 排针
- GND:  P3 排针

## PWM 参数

| 参数 | 值 |
|-----|-----|
| 频率 | 50Hz (20ms周期) |
| 0° 脉宽 | 700us |
| 120° 脉宽 | ~1633us |
| 180° 脉宽 | 2100us (理论) |

## 行为

上电后舵机从 0° 缓慢摆到 120°，再缓慢摆回来，循环往复。

## 编译

```bash
make clean all
```

## 烧录

当前使用 **SRAM 加载** 方式运行（H7RS Flash编程需额外适配）：

```bash
# 编译
make clean all

# 转为BIN
arm-none-eabi-objcopy -O binary build/servo_sram.elf build/servo_sram.bin

# 通过pyOCD加载到SRAM运行
python flash_pyocd.py
```

> 注意：SRAM运行在断电后会丢失，需要重新加载。

## 关键注意事项

1. **H7RS RCC寄存器偏移不同于标准STM32H7**：`RCC_AHB4ENR` 位于 `RCC_BASE + 0x140` (0x58024540)，而非 `+0x50`
2. 舵机必须独立5V供电，不可从开发板取电
3. 5V GND 和开发板 GND 必须连在一起（共地），否则无信号参考
4. 开发板通过 DAPLink (CMSIS-DAP) 连接电脑，用于调试和SRAM加载
