# STM32H7 智能养老监护系统

本仓库用于嵌入式系统设计竞赛项目协作，保存核心代码、接线说明、测试记录和少量必要文档。

## 项目目标

基于 STM32H7/H7RX 开发板实现居家养老监护演示闭环：

- 服药提醒：RTC 定时、蜂鸣/屏幕提示、舵机开格。
- 药量检测：HX711 + 称重传感器检测药盒余量。
- 健康监测：心率/血氧模块和六轴姿态数据采集。
- 异常报警：跌倒、心率异常或药量不足时触发告警。
- 联网展示：通过 Wi-Fi 或串口上位机展示关键数据。

## 团队分工

- 毕子扬：系统架构、接线表、文档、最终整合检查。
- 许诺言：主控软件、通信、UI、模块整合。
- 顾家铭：焊接、结构、传感器验证、模块测试记录。

## 协作规则

1. 每次开始写代码前先执行 `git pull`。
2. 单模块先放在 `firmware/module_tests/`，跑通后再合入主工程。
3. 不把教程、资料包、视频、PDF 大文件放进仓库。
4. 每次提交写清楚做了什么，例如 `完成HX711称重读取`。

## 当前保底演示链路

```text
到点提醒 -> 舵机开格 -> 称重检测余量 -> 健康/姿态异常 -> 屏幕显示 -> 告警上传
```
